# AI-Powered Financial Health Assessment for SMEs

## Problem Statement
SMEs lack simple tools to assess financial health, risks, and credit readiness.

## Solution
This platform allows SMEs to upload financial data and receive AI-driven insights, risk assessment, and financial health scores.

## Features
- CSV upload
- Financial health scoring
- Risk detection
- AI-generated insights
- Visual dashboard

## Tech Stack
- React.js
- Python (Flask, Pandas)
- AI-based insights
- Secure architecture (design-level)
